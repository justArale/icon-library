# Icon-Library

React SVG icons (for my projects). Supports custom color and size.

## Install package

```bash
$ npm i @just1arale/icons
```

## Import components

```js
// example
import { ArrorRight } from "@just1arale/icons";

<ArrorRight width="24" height="24" color="#B6D2CE">
```
